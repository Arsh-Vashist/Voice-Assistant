# Python Voice Assistant

This is a voice assistant made using Python. It listens to your voice and opens websites like YouTube, Instagram, and Google.

## Features
- Voice recognition
- Opens websites
- Speaks responses
- Tells time
- Exit command

## Requirements
Install these:

pip install pyttsx3
pip install SpeechRecognition
pip install pyaudio

## Run the program

python main.py
